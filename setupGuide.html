    설치하기

    Python설치

    python3.8.2설치

    • Windows OS(64비트) 설치


      설치시, Add Python 3.5 to PATH 체크

    온라인 환경인 경우

    pip 최신화

    pip는 파이썬 설치시, 포함되어 있습니다.

    1
    pip install --upgrade pip

    MkDocs

    MkDocs설치

    1
    pip install mkdocs

    MkDocs설치 확인

    1
    mkdocs --version

    Material for MkDocs

    1
    pip install mkdocs-material

    오프라인 환경인 경우

    패키지 다운로드

    • MkDocs
    • MkDocs-Material

    MkDocs

    mkdocs.zip파일의 압축을 C:\mkdocs에서 해제후

    1
    pip install --no-index --find-links="C:\\mkdocs" mkdocs

    Material for MkDocs

    mkdocs-material파일의 압축을 C:\mkdocs-material에서 해제후

    1
    pip install --no-index --find-links="C:\\mkdocs-material" mkdocs-materials
